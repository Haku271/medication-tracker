# 退烧药与感冒病程追踪器 - 开发完成报告

## 项目地址
https://github.com/Haku271/medication-tracker

## 技术栈
- Vue 3 + Vite
- Tailwind CSS v4
- Vue Router
- Day.js（时间处理）
- Lucide Vue（图标）/ 本项目使用 emoji

## 已完成功能

### 1. 项目脚手架
- Vue 3 + Vite 项目初始化
- Tailwind CSS v4 配置
- Vue Router 路由配置

### 2. 数据层
- localStorage 持久化
- 5 种内置退烧药配置
- coldStore 状态管理（createCold, endCold, addEntry, exportData, importData）

### 3. 诊断引擎
- 伤寒论阶段推断规则（配置驱动）
- 5 个基础阶段：太阳表证、太阳中风、少阳半表半里、阳明经证、阳明腑证
- 支持症状历史推断

### 4. 药品状态引擎
- 4 种状态：待生效、生效中、间隔期、可服用
- 实时计算距离下次可服时间
- 基于药代动力学的状态判断

### 5. 页面
- HomeView.vue：首页/详情页（自动判断有无活跃感冒）
- CreateColdModal.vue：创建感冒模态框
- AddSymptom.vue：添加症状
- AddMedication.vue：添加服药
- ExportView.vue：导出/导入
- HistoryView.vue：历史记录

### 6. 导出功能
- JSON 导出（备份恢复）
- 文字导出（给医生看，纯记录）
- JSON 导入恢复

### 7. 其他
- 自动结束（14 天后自动标记完成）
- 移动端响应式设计
- 悬浮操作按钮

## 部署

构建产物位于 `dist/` 目录。

使用 Cloudflare Pages 部署建议：
1. 在 GitHub 上创建 `Haku271/medication-tracker` 仓库
2. 将代码推送到仓库
3. 在 Cloudflare Pages 中连接 GitHub 仓库
4. 构建设置：Build command: `npm run build`，Output directory: `dist`
